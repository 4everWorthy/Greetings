# Greetings Application

## Purpose
This is a simple Spring Boot application that provides a basic REST API endpoint for greeting messages.

## How to Run
1. Clone the repository or download the project files.
2. Open the project in your preferred IDE.
3. Run the main application class (`GreetingsApplication.java`).
4. The application will start on `http://localhost:8080`.

## How to Test
### Using Postman:
1. Open Postman and create a new GET request.
2. Set the URL to `http://localhost:8080/greeting`.
3. Send the request and check the response for `"Hello, World!"`.

### Using curl:
1. Open a terminal or command prompt.
2. Run the command: `curl http://localhost:8080/greeting`.
3. Check the response for `"Hello, World!"`.
